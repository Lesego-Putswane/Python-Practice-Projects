# Simple Calculator

A basic command-line calculator built with Python.

This project allows users to perform basic arithmetic operations such as:

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## Features

- Accepts user input for numbers and operators
- Performs calculations based on the selected operator
- Handles division by zero
- Formats results (whole numbers display as integers)
- Allows repeated calculations without restarting the program

## Project Structure

The program is separated into functions for better readability and structure:

- `get_numbers()` → collects user input
- `calculator()` → handles calculation logic
- `output_format()` → formats the output
- `recalculate()` → controls repetition flow
